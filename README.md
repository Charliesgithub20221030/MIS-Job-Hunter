# MIS Job Hunter

## Install

1. Clone, create virtualenv and install `requirements.txt`

    ``` shell
    git clone ...
    cd ...
    virtualenv env
    . env/bin/activate
    pip install -r requirments.txt
    ```

2. Run server

    ``` shell
    python manage.py runserver
    ```
